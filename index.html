Cómo Hostear con GitHub Pages

¿Qué es GitHub Pages?

GitHub Pages es un servicio gratuito que te permite alojar sitios web directamente desde un repositorio de GitHub.

Paso 1: Crear un Repositorio

  1. Inicia sesión en tu cuenta de GitHub.
  2. Crea un nuevo repositorio (puedes nombrarlo nombreusuario.github.io para un sitio de usuario).

Paso 2: Agregar Archivos

  1. Sube tu archivo HTML (index.html) y otros recursos como CSS e imágenes.
  2. Si usas nombreusuario.github.io, asegúrate de que index.html sea tu archivo principal.

Paso 3: Publicar tu Sitio

GitHub Pages automáticamente publicará tu sitio. Accede a https://nombreusuario.github.io para verlo en línea.
